Asap Installation Guide

Autor Luis Uriel Garcidueñas Fraustro
Derechos Innova Solutions
Idioma en-US
Etiquetas document • installation • guide • ASAP

This guide covers the installation of ASAP cloud native, a toolkit that automates the deployment of ASAP instances using pre-configured Helm charts on your own cloud native environment. The cloud native approach significantly reduces deployment time compared to traditional methods, though there are some key operational differences to be aware of. ASAP cloud native is certified for two deployment models: a private Kubernetes cluster (general Kubernetes deployment) and Oracle Cloud Infrastructure Container Engine for Kubernetes (OKE). By following this guide, you can quickly spin up a production-ready ASAP instance on either platform.
